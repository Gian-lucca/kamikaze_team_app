# Kamikaze Team App (Flutter)

Projeto inicial em Flutter para gestão de alunos, presenças e mensalidades da Kamikaze Team.

## O que já vem pronto

- Tela de Login (mock)
- Home com painel geral
- Tela de Presenças (com lista mock e ações básicas)
- Tela de Mensalidades (com lista mock e status)
- Navegação com rotas nomeadas e BottomNavigationBar

## Como usar

1. Certifique-se de ter o Flutter instalado.
2. No seu terminal:

```bash
flutter pub get
flutter run
```

3. A partir daqui você pode:
   - Trocar mock por integrações reais com API / Firebase.
   - Adicionar novas telas conforme o escopo que planejamos.
